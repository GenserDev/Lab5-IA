# Lab5-IA — Algoritmos de Búsqueda

Implementación y comparación de algoritmos de búsqueda (DFS, BFS, Dijkstra, Greedy, A*) en laberintos y grillas.

## Instrucciones para ejecutar

### 1. Clonar el repositorio
```bash
git clone https://github.com/GenserDev/Lab5-IA.git
cd Lab5-IA
```

### 2. Crear entorno virtual
```bash
python -m venv .venv
.venv\Scripts\activate
```

### 3. Instalar dependencias
```bash
pip install numpy matplotlib jupyter
```

### 4. Ejecutar el notebook
```bash
jupyter notebook Lab5.ipynb
```
